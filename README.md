## Project Structure

For devs and contributors, the project is divided into 3 folders. Here is the logic:

### Create Submarine

This folder contains all the technical blocks and the water culling system. I registered these blocks in a dedicated section of the Create Aeronautics creative menu.

### DimensionAbyss

This concerns the dimension. All the blocks related to the abyss and the fishes are in there. Everything related to the abyss and the fishes gets its own actual dedicated creative tab. Note that there is a dedicated mixins file just for this folder.

### AbyssBiome

This folder is currently empty, but it follows the exact same logic as DimensionAbyss.
